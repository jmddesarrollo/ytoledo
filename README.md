# Y-TOLEDO CLUB DE SENDERISMO
Aplicación para la gestión de rutas de senderismo.

# Descripción
La aplicación es capaz de gestionar usuarios, permisos y rutas, con sus consultas, altas, ediciones y eliminaciones.
